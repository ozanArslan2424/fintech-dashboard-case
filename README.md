# Fintech Dashboard

## Proje Hakkında
Nodelabs vaka çalışması.

## Özellikler
- Giriş/Kayıt ekranı (form validasyonu ve toast bildirimleri)
- Dashboard (grafikler, finansal veriler, API entegrasyonu)
- Responsive tasarım
- Para birimi ve tarih formatlama

## Teknik Detaylar
- **Frontend:** Vite + React
- **Stil:** Tailwind CSS + tema dosyaları
- **State:** React useReducer, React Context API, TanStack React Query
- **API/fetch:** Axios
- **Not:** Kod içerisinde geliştirici notları eklenmiştir. "NOT:" ile başlayan yorum satırlarını aratabilirsiniz.
